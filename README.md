# Full Stack Login & Signup Application

A full-stack authentication system with user **signup** and **login** functionality.  
The frontend is built using **React (Vite)**, and the backend is built using **Node.js, Express, and MongoDB**, with secure authentication using **JWT**, **bcrypt.js**, and **Joi** for validation.
